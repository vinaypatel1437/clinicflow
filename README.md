# ClinicFlow AI Landing Page

A modern, high-converting landing page for ClinicFlow AI — AI receptionist automation for US dental clinics.

## Tech Stack

- **Next.js 14** (App Router)
- **Tailwind CSS**
- **TypeScript**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app/` — Next.js App Router pages and layout
- `src/components/` — Reusable landing page sections (Header, Hero, Trust, Problem, Solution, HowItWorks, ROI, FinalCTA, Footer)

## Design

- **Primary:** #2563EB (blue)
- **Accent:** #06B6D4 (teal)
- **Style:** Premium SaaS, Stripe/OpenAI/Notion inspired
- **Goal:** Book free demo calls for dental clinics
